How It Works

1. Data Preprocessing

Upload your forex data in CSV format. The system cleans, normalizes, and engineers features to prepare for pattern discovery.

2. Pattern Extraction

Using Template Grid approach and Dynamic Time Warping, the system identifies recurring price formations that might be overlooked by visual inspection.

3. Statistical Validation

Rigorous statistical testing ensures that identified patterns have true predictive power beyond random chance.

4. Visualization & Interpretation

Interactive visualizations help interpret discovered patterns and understand their significance in the market context.
